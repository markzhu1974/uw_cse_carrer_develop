# 从就业角度排序

如果目标是：

**2028届校招进入、腾讯、阿里、Meta、Google、Amazon等AI岗位**

我建议UW CE学生优先级如下：

| 路线          | 推荐指数  | 当前市场需求 |
| ----------- | ----- | ------ |
| ML Engineer | ⭐⭐⭐⭐⭐ | 极高     |
| Agent/LLM   | ⭐⭐⭐⭐⭐ | 极高     |
| 推荐/搜索算法     | ⭐⭐⭐⭐⭐ | 极高     |
| CV          | ⭐⭐⭐⭐  | 高      |
| Audio       | ⭐⭐⭐   | 中      |
| 硬件          | ⭐⭐⭐⭐  | 高但岗位较少 |

---

对于你目前的状态（大二结束，已学312/332/351/369），如果目标是字节、腾讯、阿里的AI校招，我认为最优解不是把自己限定在某一个方向，而是：

**大三：421 + 446 + 455/447，建立“算法 + ML + NLP/CV”的统一基础。**

然后在大三暑假前完成：

1. 一个推荐系统项目
2. 一个RAG/Agent项目
3. 一个LLM微调项目

这样到大四时，你实际上可以同时覆盖：

* ML Engineer
* Agent Engineer
* 推荐算法
* 搜索算法
* AI Platform

这五类岗位，而这五类岗位恰好占你分析的岗位需求的一半以上。
